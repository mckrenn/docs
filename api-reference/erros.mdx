## Tipos de erro
Nós usamos um padrão para os seguintes status HTTP.

### Status 401
**Motivo:** Token inválido ou não autorizado
**Ação sugerida:** Certificar-se de que o token fornecido faz parte do ambiente e o customer foi informado corretamente.

### Status 404
**Motivo:** Token não informado corretamente
**Ação sugerida:** Certificar-se de que o token foi fornecido.

### Status 405
**Motivo:** Método não permitido
**Ação sugerida:** Certificar-se de que o método foi informado corretamente.
